신시 컴퍼니 반응형 웹 사이트 제작

23.06.01. ~ 23.06.14. figma / photoshop / illustrator / html / css

뮤지컬과 공연을 기획 및 제작하는 “신시컴퍼니”의 다양한 디바이스에 적용이 가능한 반응형 웹 사이트 디자인 및 퍼블리싱 프로젝트입니다.
